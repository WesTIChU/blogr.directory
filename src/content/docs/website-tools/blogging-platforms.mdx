Tools and platforms people use to write and publish blogs — from simple hosted options to fully self-hosted setups.

---

### 🧱 Hosted Platforms (Easy to Start)

* <a href="https://wordpress.com" target="_blank" rel="noopener noreferrer"><strong>WordPress.com</strong></a> – Hosted version of WordPress, easy to get started.
* <a href="https://ghost.org" target="_blank" rel="noopener noreferrer"><strong>Ghost</strong></a> – Clean, modern publishing platform focused on writing.
* <a href="https://substack.com" target="_blank" rel="noopener noreferrer"><strong>Substack</strong></a> – Newsletter-style blogging with built-in audience tools.
* <a href="https://medium.com" target="_blank" rel="noopener noreferrer"><strong>Medium</strong></a> – Large publishing platform with built-in readership.
* <a href="https://write.as" target="_blank" rel="noopener noreferrer"><strong>Write.as</strong></a> – Minimal, privacy-focused writing platform.

---

### 🧠 Self-Hosted Platforms (More Control)

* <a href="https://wordpress.org" target="_blank" rel="noopener noreferrer"><strong>WordPress.org</strong></a> – Full control, plugins, themes, and flexibility.
* <a href="https://ghost.org/docs/install/" target="_blank" rel="noopener noreferrer"><strong>Ghost (Self-Hosted)</strong></a> – Fast, modern, and focused on content.
* <a href="https://getgrav.org" target="_blank" rel="noopener noreferrer"><strong>Grav</strong></a> – Flat-file CMS, no database required.
* <a href="https://jekyllrb.com" target="_blank" rel="noopener noreferrer"><strong>Jekyll</strong></a> – Static site generator used with GitHub Pages.
* <a href="https://gohugo.io" target="_blank" rel="noopener noreferrer"><strong>Hugo</strong></a> – Fast static site generator for blogs and sites.

---

### ⚡ Modern & Developer-Friendly

* <a href="https://astro.build" target="_blank" rel="noopener noreferrer"><strong>Astro</strong></a> – Static-first framework (what this site uses 👀).
* <a href="https://nextjs.org" target="_blank" rel="noopener noreferrer"><strong>Next.js</strong></a> – React framework for building fast websites.
* <a href="https://nuxt.com" target="_blank" rel="noopener noreferrer"><strong>Nuxt</strong></a> – Vue-based framework for modern web apps.
* <a href="https://svelte.dev" target="_blank" rel="noopener noreferrer"><strong>SvelteKit</strong></a> – Lightweight framework for fast web apps.

---

### 📝 Minimal & Indie Platforms

* <a href="https://bearblog.dev" target="_blank" rel="noopener noreferrer"><strong>Bear Blog</strong></a> – Simple, no-nonsense blogging platform.
* <a href="https://micro.blog" target="_blank" rel="noopener noreferrer"><strong>Micro.blog</strong></a> – Indie blogging and micro-posting platform.
* <a href="https://blot.im" target="_blank" rel="noopener noreferrer"><strong>Blot</strong></a> – Turns a folder into a blog.
* <a href="https://listed.to" target="_blank" rel="noopener noreferrer"><strong>Listed</strong></a> – Blogging powered by Standard Notes.

---

### Why these platforms?

These are tools used by real people writing real blogs — not content farms or mass-produced sites.

Some are simple, some are powerful, but all of them allow people to publish their own voice on the web.
