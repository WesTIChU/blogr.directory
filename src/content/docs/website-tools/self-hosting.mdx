### Self-Hosting Platforms

* <a href="https://coolify.io" target="_blank" rel="noopener noreferrer"><strong>Coolify</strong></a> – Self-hosted alternative to Vercel and Heroku with a clean UI.
* <a href="https://caprover.com" target="_blank" rel="noopener noreferrer"><strong>CapRover</strong></a> – Self-hosted alternative to Heroku with simple app deployments.
* <a href="https://dokku.com" target="_blank" rel="noopener noreferrer"><strong>Dokku</strong></a> – Lightweight, self-hosted Heroku-style platform.
* <a href="https://www.portainer.io" target="_blank" rel="noopener noreferrer"><strong>Portainer</strong></a> – Self-hosted UI for managing Docker containers. <Badge text="DOCKER" size="small" />
* <a href="https://umbrel.com" target="_blank" rel="noopener noreferrer"><strong>Umbrel</strong></a> – Personal server platform, similar to an app store for self-hosting.
* <a href="https://casaos.io" target="_blank" rel="noopener noreferrer"><strong>CasaOS</strong></a> – Beginner-friendly home server dashboard, similar to Umbrel.

---

### Containers & Infrastructure

* <a href="https://www.docker.com" target="_blank" rel="noopener noreferrer"><strong>Docker</strong></a> – Runs apps in containers, like lightweight virtual machines.
* <a href="https://kubernetes.io" target="_blank" rel="noopener noreferrer"><strong>Kubernetes</strong></a> – Advanced system for managing Docker containers at scale.
* <a href="https://www.linuxserver.io" target="_blank" rel="noopener noreferrer"><strong>LinuxServer.io</strong></a> – Prebuilt Docker containers for common self-hosted apps.

---

### Networking & Access

* <a href="https://tailscale.com" target="_blank" rel="noopener noreferrer"><strong>Tailscale</strong></a> – Private network (VPN) that lets you access your server from anywhere.
* <a href="https://nginxproxymanager.com" target="_blank" rel="noopener noreferrer"><strong>Nginx Proxy Manager</strong></a> – Easy UI for routing domains to your apps.
* <a href="https://traefik.io" target="_blank" rel="noopener noreferrer"><strong>Traefik</strong></a> – Advanced reverse proxy for dynamic setups.
* <a href="https://www.cloudflare.com" target="_blank" rel="noopener noreferrer"><strong>Cloudflare</strong></a> – Adds DNS, security, and CDN in front of your server.

### Why self-host?

Self-hosting gives you full control over your websites and data.

It’s more hands-on than traditional hosting - but it lets you build, experiment, and own your corner of the web completely.
