Tools that help you run your own websites, apps, and services without relying on traditional hosting platforms.

### Self-Hosting Platforms

* <a href="https://coolify.io" target="_blank" rel="noopener noreferrer"><strong>Coolify</strong></a> – Self-hosted alternative to Vercel and Heroku with a clean UI.
* <a href="https://caprover.com" target="_blank" rel="noopener noreferrer"><strong>CapRover</strong></a> – Self-hosted alternative to Heroku with simple app deployments.
* <a href="https://dokku.com" target="_blank" rel="noopener noreferrer"><strong>Dokku</strong></a> – Lightweight, self-hosted Heroku-style platform.
* <a href="https://umbrel.com" target="_blank" rel="noopener noreferrer"><strong>Umbrel</strong></a> – Personal server platform with an app-store style experience.
* <a href="https://casaos.io" target="_blank" rel="noopener noreferrer"><strong>CasaOS</strong></a> – Beginner-friendly home server dashboard.

### Containers & Infrastructure

* <a href="https://www.docker.com" target="_blank" rel="noopener noreferrer"><strong>Docker</strong></a> – Runs apps in containers. <Badge text="CORE" size="small" />
* <a href="https://www.portainer.io" target="_blank" rel="noopener noreferrer"><strong>Portainer</strong></a> – UI for managing Docker containers. <Badge text="DOCKER" size="small" />
* <a href="https://kubernetes.io" target="_blank" rel="noopener noreferrer"><strong>Kubernetes</strong></a> – Container orchestration at scale.
* <a href="https://www.linuxserver.io" target="_blank" rel="noopener noreferrer"><strong>LinuxServer.io</strong></a> – Prebuilt Docker containers for common apps.

### Networking & Access

* <a href="https://tailscale.com" target="_blank" rel="noopener noreferrer"><strong>Tailscale</strong></a> – Secure remote access from anywhere.
* <a href="https://nginxproxymanager.com" target="_blank" rel="noopener noreferrer"><strong>Nginx Proxy Manager</strong></a> – Easy reverse proxy with SSL.
* <a href="https://traefik.io" target="_blank" rel="noopener noreferrer"><strong>Traefik</strong></a> – Advanced reverse proxy for dynamic setups.
* <a href="https://www.cloudflare.com" target="_blank" rel="noopener noreferrer"><strong>Cloudflare</strong></a> – DNS, security, and CDN in front of your server.

## Why self-host?

Self-hosting gives you more control over your apps, files, and data.

It takes more effort than managed hosting, but gives you flexibility, ownership, and room to experiment.
