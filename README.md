# BlogR Directory

> A curated collection of human-written websites, blogs, and real corners of the internet.

---

## 🌐 About

BlogR Directory is a hand-curated collection of **human-written blogs, personal websites, and genuine online communities**.

No AI sludge.  
No recycled SEO filler.  
No content farms pretending to be useful.  

Just real people writing real things.

This isn’t trying to be the biggest directory on the internet.  
It’s trying to be a good one.

---

## ✨ What makes it different

- Every site is found manually  
- Every link is checked manually  
- No automated scraping or bulk imports  
- Focus on quality over quantity  
- Built around discovery, not algorithms  

If it doesn’t feel human, it doesn’t go on the site.

---

## 🧱 Built with

- Astro → https://astro.build  
- Starlight → https://starlight.astro.build  
- Markdown  

Simple, fast, and no unnecessary bloat.

---

## 🚀 Project Structure

    .
    ├── public/
    ├── src/
    │   ├── assets/
    │   ├── components/
    │   ├── content/
    │   │   └── docs/
    │   └── styles/
    ├── astro.config.mjs
    ├── package.json
    └── tsconfig.json

- Content lives in `src/content/docs/`  
- Each `.md` or `.mdx` file becomes a page  
- Images go in `src/assets/` or `public/`  

---

## 🧞 Commands

Run from the root of the project:

- npm install → Install dependencies  
- npm run dev → Start dev server (localhost:4321)  
- npm run build → Build site to `/dist`  
- npm run preview → Preview production build

## 🤝 Contributing

Know a site worth adding?

You can:
- Submit a site via GitHub issues  
- Edit pages and open a pull request  

All submissions are reviewed manually.

### What belongs here

- Personal blogs  
- Independent writers  
- Small communities  
- Thoughtful niche websites  

### What doesn’t

- AI-generated content farms  
- Affiliate spam  
- Generic SEO sites  
- Mass-produced content  

---

## 🤖 AI & content

AI tools are fine.

They help people build, design, and get online — and that’s a good thing.

But the content itself should be human.

Real thoughts.  
Real writing.  
A real voice behind it.

---

## 💭 Why this exists

The modern internet is full of noise.

This is an attempt to keep a small part of it:
- slower  
- more personal  
- actually worth exploring  

---

## 🙌 Final note

This isn’t about building the biggest list.

It’s about building one that still feels like it was made by a person.
